# Handwritten Digit Recognition (Deep Learning)

## Objective
Recognize digits (0–9) using Deep Learning.

## Tools Used
- Python
- TensorFlow / Keras
- Matplotlib

## Steps
1. Load dataset (MNIST)
2. Normalize data
3. Build neural network
4. Train model
5. Evaluate accuracy
6. Predict digits

## Output
Model predicts handwritten digits with high accuracy.
